# 🧮 Minha Calculadora

Uma calculadora simples feita em **HTML, CSS e JavaScript**, com suporte a:
- Operações básicas (+, -, ×, ÷)
- Limpar e apagar último dígito
- Exibir o símbolo de multiplicação como **×**
- Digitação pelo teclado físico (Enter, Backspace, Escape)
